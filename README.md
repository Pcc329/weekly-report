# 📊 專案進度週報看板 (Weekly Status Dashboard)

這是一個自動化的週報發布系統，旨在透過單一、永久的網頁連結，為團隊提供即時、透明的專案進度資訊。

## 🌟 核心特色

- **單一來源 (Single Source of Truth)**：同事只需存取此網頁，即可看到最新的進度報告，無需頻繁下載 Word 附件。
- **自動同步更新**：透過內嵌 Google Doc 發布技術，管理員只需編輯 Google 文件，網頁內容將自動同步更新。
- **響應式設計**：支援電腦與手機瀏覽，方便隨時隨地查看。
- **輕量化架構**：無需後端資料庫，維護成本極低。

## 🛠 運作原理

1.  **內容層 (Content)**：使用 Google Doc 撰寫每週進度（包含本週達成目標、下週規劃、獲獎紀錄等）。
2.  **發布層 (Publishing)**：透過 Google Doc「發布到網路」功能產生嵌入連結。
3.  **展示層 (Presentation)**：本專案的 `index.html` 透過 `<iframe>` 讀取發布連結，並佈署於 **GitHub Pages** 提供穩定訪問。

## 📅 更新頻率

- **發布時間**：每週五中午前 (Friday Release)。
- **內容範圍**：包含資服業者資料庫清理進度、AI 自動補齊狀態、得獎名單比對結果等。

## 🚀 如何使用

1. **對於管理員**：
   - 直接開啟關聯的 Google Doc 進行編輯。
   - 存檔後，變更將在幾分鐘內自動反映在此網頁上。
   
2. **對於團隊成員**：
   - 請將本頁面網址 [在此輸入您的 GitHub Pages 網址] 加入瀏覽器書籤。
   - 若內容未更新，請嘗試重新整理網頁。

## 📁 檔案說明

- `index.html`: 主要的入口網頁，包含 CSS 樣式與 Google Doc 嵌入連結。
- `README.md`: 本說明文件。
