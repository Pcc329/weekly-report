| 產業 | 餐飲、製造、零售、醫療、物流、建築營造、金融、教育、旅遊住宿、電商 |
| 需求 | 解決人力短缺、提升辦公室效率、加強資訊安全、簡化現金流、接觸新客戶、改善客戶體驗、簡化供應鏈、淨零碳排、吸引人才、數位轉型入門 |
| 測試表 | Test_Search_Results、Test_Ask_Results |

### 搜尋流程（修正後）

```
查詢語句
→ POST /api/claude（Claude 解析語意，回傳篩選條件）
→ 用篩選條件過濾快取中的方案
→ 回傳相關結果
→ POST /api/ask（帶入搜尋結果，Claude 生成回答）
```
